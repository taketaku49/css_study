詳解CSS基礎文法編
  1. スタイルシートを別ファイルとする　＃３
  2. HTML内にスタイルを書く
  3. スタイルの継承　＃５
    　colorは継承されるが、borderは継承されない。mdnというサイトで調べる。 しかしinheritとすると継承される。
  4. テキストにスタイルを設定しよう　＃８
    　打消し線、下線、リンクの下線を消す。text-decoration
  5. font-familyでフォントを指定しよう　＃９
    　フォントの指定で日本語フォントを先に書くと、 英字に日本語のフォントが適用されてしまうので、英文フォントのあとに書くとよい。 sans-serifは総称フォントファミリーと呼ばれていて、個々のフォント名ではなくてフォントのカテゴリーを指定するものになる。 sans-serif=ゴシック、serig=明朝、cursive=筆記体、monospace=等幅フォント
  6. line-heightで行の高さを調整しよう　＃１０＃１１
    指定は48px、3em、3 といった単位。3emは3文字分。3は指定されているフォントの3倍。継承された場合に少し違いがでる。
  7. vertical-alignで位置を調整しよう　＃１２
    vertical-align:middleとか。top,bottom,baseline,2pxなど。
  8. RGBAを使った色の表現方法を覚えよう　＃１３＃１４
    色キーワード　color: red;
    RGBAによる指定　Red(0-255) Green(0-255) Blue(0-255) Alpha(0.0-1.0) color: rgba(255,0,0,1.0);　16進 color:#ff0000ff; = color:#f00f;
    Alphaを指定しない場合　rgb(0,0,255);
    HSLAによる指定　Hue(色相0から360),Saturation(彩度。パーセントで指定),Lightness(明度。パーセントで指定),Alpha(透明度 0.0-1.0)
    color:hsla(180,80%,40%,1.0); = hsl(180,80%,40%);
    要素全体の透明度の指定は opacity:0.5; のように指定。
  9. リストをスタイリングしてみよう　＃１５＃１６
    list-style-type: circle;　白マルになる
    list-style-position: inside;　点が内側になる
    list-style-image: url(icon.png);　記号の変わりにイメージになる
    list-style:circle inside url(icon.png);　一括指定の書き方。上の3行と同じになる。
    list-style-type: lower-alpha;　a,b,cになる
  10. クラスセレクターの基本　＃１７
    p class="info"　.info{。。。}
  11. ボックスモデルを理解しよう　＃１８
    コンテンツの幅と高さはwidthとheightプロパティ
    境界線をつけたい場合はborderプロパティ、borderの内側の余白はpadding、borderの外側はmarginプロパティ
  12. Width、heightを設定してみよう　＃１９
    width:50%;とすると親要素の50%になる。
    あふれた場合の指定は overflow:hidden;とするとそこまでしか表示せず
    overflow:scroll;とするとスクロールバーが表示される。
  13. borderで境界線をつけてみよう　＃２０
    border-with:2px; border-style:solid; border-color:brue; 一括指定では border:2px solid blue;と書ける。
    solidは実線。dashedは点線。
    border-top,border-bottomのように一部のみ指定も可能。noneとすると設定なしとなる。
    角を丸くするには border-radius: 10px;のようにする。
  14. paddingで内側の余白をつけよう　＃２１
    padding-top: 20px; padding-left: 40px;　
    一括指定では padding: 10px 20px 30px 40px;のように書く
  15. marginで外側の余白をつけよう　＃２２
    margin: 0 0 20px; /* 下に余白が20pxできる */　
    margin-left: auto; /* 左に余白 */　
    margin-right: auto; /* 右に余白。上と合わせて結果的に中央となる */
  16. displayプロパティについて理解しよう　＃２５
    display: inline; /* 左につめたい inlineにするとwidthとheightはきかなくなる */　
    display: inline-block;　/* width、heightが反映される */
    display: none; こうすると消えてしまう
  17. positionで要素の位置を変えよう　＃２６
    position: relative; /* staticの位置からずらして配置したい場合 */
    top:30px; /* staticの位置から30px分移動する */
    left:30px;
    position: fixed; /* windowの左上を起点にする */
    top:30px; /* 左上の位置から30px分移動する */
    left:30px;
  18. 要素を絶対配置してみよう　＃２７
    要素が親子関係の時
    position: absolute; /* 絶対配置 */
    /* 親要素がstaticであれば左上を起点とする */
    /* 親要素がstatic以外(relativeとか)であれば親要素を起点とする */
    /* top: 30px;
    left: 10px; */
    top: -5px;
    right: -5px;
  19. z-indexで重なり順を操作しよう　＃２８
    z-index: 3;　数値の大きいのが前にくる　
    つけなければ、後に書いたものが重なっていく
  20. box-sizingを使ってみよう　＃２９
    widthとheightにはpaddingとborderが含まれていないが
    場合によってはこれらを含めたい場合がある　
    box-sizing: border-box;とするとwidthとheightが
    paddingとborderを含めるようになる。 つけなければ、後に書いたものが重なっていく
  21. calcで単位が違う計算をしてみよう　＃３０
    単位が違っても計算できる。 + - * / を使う。演算子の前後は半角ブランクが必要
  22. 要素に影をつけてみよう　＃３１
    box-shadow: 10px 5px rgba(0,0,0,0.3); /* ボックスのX方向、Y方向に薄い影をつける */
    text-shadow: 10px 5px rgba(0,0,0,0.3); /* 文字にもつける */
    box-shadow: 10px 5px rgba(0,0,0,0.3); /* 3番目の数値をつけて、影のぼかし具合を設定 */
    box-shadow: 10px 5px 3px 10px rgba(0,0,0,0.3); /* ４番目で影を拡大する */
  23. 背景のスタイリングをしていこう　＃３２
    background-image: url(../img/header.png);
    background-size: cover; /* 画像の縦横比を保持したまま、なるべく大きく表示したい場合 */
    background-position: center; /* デフォルトは画像の左上が起点。画像の中心を起点にしたい場合 */
    /* 一括指定 */
    background: center/cover pink url(../img/header.png);
  24. floatを使ってみよう　＃３３
    img{ float: right; } /* イメージを右に寄せる 通常の配置から外れる */
    クリアする場合は
    clear: right;
