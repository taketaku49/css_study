  1. CSSの基本セレクターを学んでいこう　＃１ セレクターのパターン
  2. 要素型セレクター､classセレクターを使おう　＃２ HTMLにclass="name1 name2"のように複数つけることも可能。
  3. idセレクター､全称セレクターを使おう　＃３ 
          <h1 id="title">見出し</h1> 
      #title {
        color:pink;
      }
      id属性は同じページ内で同じ値を使ってはいけない。
      また、空白もダメなのでclass属性のように複数指定はできない。
      したがって、柔軟なスタイリングをするならclass属性のほうが良いかも。
      
      全称セレクタ
      * {
        font-size: 12px;
      }
  4. 属性セレクターを使ってみよう　＃４
    5.           cssの書き方
          存在の有無　　[attr]
          完全一致　　　[attr="value"]
          前方一致　　　[attr^="value"]
          後方一致　　　[attr$="value"]
          部分一致　　　[attr*="value"]
