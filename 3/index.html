詳解CSSフレックスボックス編
  1. フレックスボックスを使ってみよう　＃１ 
      HTMLで親要素のdivにクラス(container等)を指定し、
  CSSでは
  .container {
    display: flex;  /* フレックスボックスを使うという指定 */
  }
  2. 関連用語について理解しておこう　＃２
    Flex ContainerとFlex Item
  3. 要素の並び､揃えを制御しよう　＃３
    flex-directionとjustify-contentとalign-items
  4. flex-wrap､align-contentを使おう　＃４ flex-wrapとalign-content
  5. align-self､orderを使ってみよう　＃５ align-selfとorder
  6. flex-basisでサイズを決めよう　＃６
    flex-grow,flex-shrinkでの計算に使われる要素のサイズを決めるためのプロパティ。
  7. flex-grow､flex-shrinkを理解しよう　＃７
    flex-growは余っているときの振り分け割合、flex-shrinkは足りない時に縮める割合。
  8. flexプロパティを使ってみよう　＃８
    一括指定が可能
