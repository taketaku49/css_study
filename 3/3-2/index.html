Flex Container

  • flex-direction　主軸の方向を決める。デフォルトは左から１，２，３
  • justify-content　主軸に沿って要素をどう揃えるかを指定。デフォルトは左揃え。space-betweenは要素の間に余白が均等に配置される。
  • align-items　交差軸に沿って要素をどう揃えるか。縦方向の位置。デフォルトは上揃え。flexboxでalignとなっていたら交差軸に対しての操作。
  • flex-wrap　要素の幅がコンテナの幅を超える場合のプロパティ。デフォルトは折り返さずにそれぞれの幅が縮む。
  • align-content　flex-wrap:wrapになっている時に交差軸方向で要素を揃えたい時に使う。

Flex Item・・・個々のItemについて

  • align-self　交差軸方向の揃え
  • order　順番
  • flex-basis　個々のアイテムをどれだけ伸ばしたり縮めたりするかを指定するプロパティ。
    　widthで設定したよりもflex-basisで指定した値が有効。max-widthが指定されていればそれが有効となる。
  • flex-grow　余白の余った分の割り当てをする
  • flex-shrink　幅が足りなかった時にどの要素を縮めるかを指定する
