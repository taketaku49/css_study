flex-direction:row・・主軸の方向を決めるためのプロパティ。デフォルトはrowで左から１，２，３。
flex-direction:row-reverse・・右から１，２，３
flex-direction:column・・上から１，２，３
flex-direction:column-reverse・・下から１，２，３
justify-content:flex-start・・・主軸に沿って要素をどう揃えるかを指定。これは左揃え。
justify-content:flex-end・・・主軸に沿って右揃え。
justify-content:center・・・主軸に沿って中央揃え。
justify-content:space-between・・・要素の間に余白が均等に配置。
align-items:flex-start・・・上揃え。
align-items:flex-end・・・下揃え。
align-items:center・・・中央揃え。
